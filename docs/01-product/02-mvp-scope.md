# MVP Scope

**Version:** 1.0
**Status:** Approved
**Last Updated:** 05 July 2026
**Owner:** Steps Tech Ltd

## Purpose
Deliver a production-ready MVP of Apex Card for Diaspora Invest.

## Geographic Scope
- Europe (including the UK)
- Cape Verde

## Languages
- English
- Portuguese

## Platforms
- React Native mobile app (iOS & Android)
- Partner interface (same app)
- React Admin Portal
- Express REST API

## Included Features
- Authentication
- User profile
- Digital membership with QR code
- Partner directory
- Search by country/category/name
- Favourite partners
- Reviews & ratings
- Partner QR validation
- Admin portal
- User/Partner/Country/Category CRUD
- RevenueCat subscriptions
- Basic analytics
- Cloud deployment

## Excluded from MVP
- Airport transfers
- Bookings
- Loyalty
- Referral programme
- CRM
- AI recommendations
- Commission engine
- Stripe service payments
