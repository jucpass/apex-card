# User Personas

## Member – Traveller
- Average age: ~45
- Motivation: Save money while travelling.
- Technical confidence: Average.
- Regions: Europe and Cape Verde.

## Member – Resident
- Students and local residents.
- Younger demographic.
- Motivation: Everyday savings.
- Technical confidence: Average to high.

## Partner
- Registered small businesses.
- Sectors: Restaurants, Hotels, Shopping, Education, Travel, Beauty and Health.
- Goal: Acquire new customers and build loyalty.

## Administrator
Permissions assigned individually rather than fixed roles.
Examples:
- Users
- Partners
- Reviews
- Countries
- Categories
- Analytics
- Financial
- Create / Update / Delete / Suspend
